# git-course-task

This repository was created for a GitHub course assignment.

## Project Description
This project includes a simple HTML page and demonstrates the basic GitHub workflow:
- Creating a repository
- Using branches
- Making commits
- Opening a Pull Request
- Merging changes into `main`

## Files
- `README.md`
- `index.html`

## Branches
This repository includes:
- `main`
- `feature-readme`

## GitHub Workflow
The following steps were completed in this project:
1. Created a repository
2. Created an additional branch
3. Made changes to the project files
4. Committed the changes
5. Opened a Pull Request
6. Merged the changes into `main`

## Purpose
The purpose of this repository is to practice working with GitHub branches, commits, pull requests, and merges.
